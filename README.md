# WEB1Aula02

Nesta aula, aprendemos sobre a utlizar algumas tags importantes, como: h1 para título, p para parágrafo, img para adicionar imagem, e tags para listas, como: ul para não ordenada, ol para ordenada, dl para definição, li para listar os itens da lista, dt para identificar um termo e dd para definição do termo.
