Definição de Tags da aula 2 de WEB1:

  • h1
    Marca o que é título
  • p
    Marca o que são parágrafos
  • img
    Tag para inserir uma imagem
  • Src
    Onde insere o endereço da imagem
  • ul
    Para listas não ordenadas
  • ol
    Para listas ordenadas
  • dl
    Para listas de definição (como esta)
  • li
    Para listas de itens
  • dt
    Identifica um termo na lista de definição
  • dd
    Fornece detalhes ou a definição mais completa do termo
